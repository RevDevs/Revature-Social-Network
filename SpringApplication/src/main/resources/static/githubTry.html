    Github user Scopes
  • (no scope) -> Grants read-only access to public information (includes public user profile info, public repository info, and gists)
  • user -> Grants read/write access to profile info only. Note that this scope includes user:email and user:follow.
  • user:email -> Grants read access to a user's email addresses. user:follow Grants access to follow or unfollow other users.
